# MovieCat

![badge](https://img.shields.io/github/last-commit/mbabicz/MovieCat?style=for-the-badge)



## ABOUT THE PROJECT
Movie search application that allows you to get detailed information about the movie, cast, funding, youtube trailers and reviews. It has its own database, which allows the user to write reviews. The application allows you to log in, register or quickly set up a guest account. The guest account has some limitations, but the most important features remain active.
MovieCat uses the [TMDB API](https://developers.themoviedb.org/3) as movie database. The connection of this API with the application made thanks to [this guide](https://www.youtube.com/watch?v=cTNhMgNV53s).
User database was created on the Firebase platform. The app is developed with SwiftUI using MVVM pattern.

## API KEY
To use this application please enter you API KEY in *MovieStore file* in the field marked: *"API_KEY"* <br />
The API KEY is necessary to run this application. It can be generated on the [TMDB website](https://www.themoviedb.org/settings/api). <br />
